# 🔑 Local Passport Auth Service

## 📌 Mô tả
Dự án minh họa **cơ chế đăng ký, đăng nhập, session** bằng Node.js + Passport.  
Bạn có thể test API bằng **Postman** hoặc form HTML.

Các tính năng:
- Đăng ký người dùng (`/auth/register`)
- Đăng nhập (`/auth/login`)

---

## 🚀 Cài đặt & chạy
-npm install 
-node app.js
